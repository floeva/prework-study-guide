Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The img element is used to add an image in html
  • The title element is used to display a title of the document

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations contain the property and the value
  • A stylesheet requires a link to the relevant html file
  • Adding a class attribute as a selector allows you to assign elements to the new rule
  • A external style sheet keeps all CSS rules in an separate external file
  • Group selectors to one CSS rule with a comma for DRY code

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add [file]: adds a file
  • git commit: creates a new commit from new changes made
  • git push: pushes local changes to the repo
